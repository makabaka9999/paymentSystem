# 优选商城 MVP 产品需求文档

## 1. 项目定位

优选商城是一个原创中文 B2C/C2C 多商家电商平台 MVP，服务普通消费者、入驻商家和平台运营管理员。平台参考综合电商常见购物链路，但不使用任何既有平台的品牌、素材、文案或相同界面。

## 2. 目标用户

- 消费者：搜索、浏览、收藏、加购、下单、模拟支付、评价商品。
- 商家：维护店铺信息，发布和管理商品，处理订单。
- 管理员：管理用户、商家、商品审核、订单和平台数据统计。

## 3. MVP 范围

- 用户端：首页、商品列表、商品详情、登录注册、购物车、订单确认、我的订单、收藏、地址和资料。
- 商家端：商品发布、商品管理、订单管理、店铺信息管理。
- 后台端：用户管理、商家管理、商品审核、订单管理、数据统计。
- 交易能力：搜索、分类、购物车、优惠券、下单、订单状态、模拟支付。

## 4. 核心流程

1. 用户进入首页，浏览分类、推荐商品、热门店铺和限时优惠。
2. 用户通过搜索或分类进入商品列表，按价格、销量、评分筛选排序。
3. 用户进入商品详情，选择 SKU 与数量，加入购物车或立即购买。
4. 用户在购物车勾选商品并调整数量，进入订单确认。
5. 用户选择地址、优惠券、配送方式，提交订单，订单状态为待付款。
6. 用户执行模拟支付，订单状态变为待发货，后续可扩展发货、收货、评价。
7. 商家发布商品，平台后台审核后对消费者可见。

## 5. 功能需求

### 用户端

- 账号：手机号或邮箱登录、密码登录、注册、基础表单校验、JWT 会话。
- 商品：搜索、分类、价格筛选、销量排序、评分排序、详情、SKU、库存展示。
- 购物车：加入购物车、勾选、改数量、删除、总价计算。
- 订单：确认地址、商品清单、优惠券、配送方式、提交订单、模拟支付。
- 个人中心：我的订单、我的收藏、收货地址、个人资料。
- 互动：收藏商品、评价商品。

### 商家端

- 店铺信息：店铺名称、简介、类目、评分、状态。
- 商品发布：标题、分类、价格、原价、库存、SKU、图片、详情。
- 商品管理：上下架、库存、价格、审核状态。
- 订单管理：查看订单、发货、售后入口预留。

### 后台管理端

- 用户管理：查看、禁用、角色维护。
- 商家管理：入驻审核、状态管理。
- 商品审核：审核通过、驳回、违规处理。
- 订单管理：全平台订单查询、状态追踪。
- 数据统计：成交金额、订单数、转化率、待审核数。

## 6. 非功能需求

- 响应式布局：适配桌面和手机端。
- UI 风格：橙色、白色、浅灰色为主，现代、清爽、原创。
- 安全：密码加密存储、JWT 鉴权、角色权限控制。
- 性能：列表分页、图片懒加载、搜索条件索引。
- 可扩展：后续接入真实支付、物流、售后、推荐系统和风控。
